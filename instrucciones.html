⚙️ Instrucciones de configuración

Seguí estos 5 pasos para dejar el asistente funcionando en tu hosting.

1

Crear la Google Sheet

Entrá a sheets.google.com y creá una hoja nueva. Podés llamarla "Config Biblioteca". No necesitás escribir nada adentro, el script lo hace solo.

2

Abrir el editor de Apps Script

Dentro de la Google Sheet, hacé clic en el menú Extensiones → Apps Script. Se abre el editor de código.

3

Pegar el código del script

Borrá todo el contenido que hay en el editor y pegá este código:

function doGet(e) {
  var action = e.parameter.action;
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();

  if (action === 'get') {
    var apiKey = sheet.getRange('A1').getValue();
    var instructivo = sheet.getRange('B1').getValue();
    if (!apiKey || !instructivo) {
      return jsonResponse({ ok: false, error: 'Sin configuración' });
    }
    return jsonResponse({ ok: true, apiKey: apiKey, instructivo: instructivo });
  }

  return jsonResponse({ ok: false, error: 'Acción no válida' });
}

function doPost(e) {
  var body = JSON.parse(e.postData.contents);
  if (body.action === 'save') {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    sheet.getRange('A1').setValue(body.apiKey);
    sheet.getRange('B1').setValue(body.instructivo);
    return jsonResponse({ ok: true });
  }
  return jsonResponse({ ok: false, error: 'Acción no válida' });
}

function jsonResponse(obj) {
  return ContentService
    .createTextOutput(JSON.stringify(obj))
    .setMimeType(ContentService.MimeType.JSON);
}
4

Publicar como aplicación web

En el editor de Apps Script:

1. Hacé clic en Implementar → Nueva implementación

2. En "Tipo" seleccioná Aplicación web

3. En "Ejecutar como" elegí Yo

4. En "Quién tiene acceso" elegí Cualquier persona

5. Hacé clic en Implementar y autorizá los permisos

6. Copiá la URL que aparece — empieza con https://script.google.com/macros/s/...

⚠️ Guardá bien esa URL, la vas a necesitar en los pasos siguientes.
5

Configurar los archivos HTML

En admin.html: pegás la URL del script en el campo correspondiente. Desde ahí cargás el instructivo y la API key.

En consulta.html: abrí el archivo con un editor de texto y buscá esta línea:

const SCRIPT_URL = 'PEGAR_URL_DEL_SCRIPT_AQUI';

Reemplazá PEGAR_URL_DEL_SCRIPT_AQUI con tu URL del script.

Luego subí los tres archivos a tu hosting y listo. Los usuarios acceden a consulta.html y vos administrás desde admin.html.
